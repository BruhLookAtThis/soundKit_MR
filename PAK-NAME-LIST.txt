# (2) The name of the folder that will contain your pak file, output into 2-FINAL-PAK-FILES-ARE-HERE
# (3) The name of the pak file inside of the (2) folder
# Below is an example of what to type when filling this list out:
# BNK_FILENAME:FOLDER_NAME_YOU_WANT:PAK_FILE_NAME_YOU_WANT_99999999_P
